# Baremetal NLP

> Natural language processing from first principles using phi-harmonic mathematics

## Usage

Use this skill to perform natural language processing without external dependencies. All operations are built from first principles using phi-harmonic tokenization, embedding, and semantic analysis.

## Syntax

```
TOOL:NLP <operation> <parameters>
```

## Operations

- `tokenize` - Phi-harmonic boundary detection
- `embed` - Phi-harmonic word embedding
- `similarity` - Phi-harmonic semantic similarity
- `understand` - Phi-harmonic language understanding
- `synthesize` - Phi-harmonic text synthesis

## Examples

```
TOOL:NLP tokenize text="Hello world"
TOOL:NLP embed word="hello"
TOOL:NLP similarity text1="cat" text2="dog"
TOOL:NLP understand text="The quick brown fox"
TOOL:NLP synthesize concept="joy"
```
