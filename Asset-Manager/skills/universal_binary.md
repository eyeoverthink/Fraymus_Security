# Universal Binary Converter

> Language-independent binary processing using phi-harmonic mathematics

## Usage

Use this skill to convert any file to binary and back, enabling language-independent processing at the nervous system level of computing.

## Syntax

```
TOOL:BINARY <operation> <parameters>
```

## Operations

- `to_binary` - Convert any file to binary (language-independent)
- `from_binary` - Universal reconstruction (lossless)
- `analyze` - Phi-harmonic binary analysis (ones/zeros, significance)
- `compress` - Phi-harmonic compression (run-length encoding with φ threshold)
- `decompress` - Binary decompression (lossless)
- `learn_pattern` - Binary-level pattern learning
- `recognize_pattern` - Binary pattern recognition (language-independent)

## Examples

```
TOOL:BINARY to_binary file=data.txt
TOOL:BINARY from_binary binary=01001000...
TOOL:BINARY analyze binary=01001000...
TOOL:BINARY compress binary=01001000...
TOOL:BINARY learn_pattern binary=01001000...
TOOL:BINARY recognize_pattern binary=01001000...
```

## Capabilities

- Language-independent conversion (any file → binary)
- Phi-harmonic binary pattern analysis
- Lossless compression with phi-threshold
- Binary-level pattern learning
- Language-independent pattern recognition
- Universal formatting support

## Philosophy

Binary is the universal language of computing - the nervous system level. By converting any file to binary, we can abstract at the lowest possible level (the nervous system), learn on a low level (less memory, less leakage), execute quicker (direct binary processing), leave no traces (pure binary operations), and achieve language independence (everything becomes binary).

## Foundation

Built using phi-harmonic mathematics (φ = 1.618033988749895) as the foundation for all operations.

## Safety

This tool is safe to use and runs in a browser-based environment with zero external dependencies.
